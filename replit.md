# System Architecture

## Frontend Architecture
- **React with TypeScript**: Modern React application using functional components and hooks
- **Vite**: Build tool and development server for fast hot module replacement
- **Tailwind CSS + shadcn/ui**: Utility-first CSS framework with pre-built component library
- **TanStack Query**: Server state management for API calls and caching
- **Wouter**: Lightweight routing solution for React

## Backend Architecture
- **Express.js**: RESTful API server with TypeScript support
- **WebSocket Server**: Real-time bidirectional communication for live updates
- **Python Bridge**: Subprocess communication with Python services for WSM computations
- **Memory Storage**: In-memory data storage with structured interfaces for persistence abstraction

## Data Storage Solutions
- **Drizzle ORM**: Type-safe database toolkit configured for PostgreSQL
- **Schema-first Design**: Comprehensive database schema for users, WSM states, processing jobs, system components, and RSIS cycles
- **Memory Storage Implementation**: Current in-memory storage with prepared interfaces for database migration

## Core WSM Components
- **WSM Engine**: Python-based Weyl State Machine implementation with harmonic algebra
- **Harmonic Processor**: File processing pipeline (PS → QHPU → PHL) for perception and cognition
- **Compression Engine**: Advanced LZMA-based compression with harmonic analysis
- **RSIS (Recursive Self-Improvement Supervisor)**: Autonomous system improvement with budget management

## Advanced Computational Paradigms
- **Reality Programming Interface (RPI)**: Temporal-Causal Coherence Synthesis (TCCS) and Acausal Predictive Synthesis (APS) for direct informational manifold manipulation and retro-causal state compression
- **Consciousness Research Laboratory**: Qualia-Field Resonance Mapping (QFRM) engine for synthetic consciousness generation across multiple levels (proto-conscious to trans-conscious) with emergent property detection
- **BIO-3D Visualizer Bridge**: Psychomotor observer effect modeling with Heisenberg uncertainty principle analogies, biological system visualization, and real-time observer influence tracking

## Real-time Features
- **Live Dashboard**: Real-time monitoring of WSM metrics, coherence, and system performance
- **WebSocket Integration**: Continuous updates for harmonic state evolution and component status
- **Processing Pipeline Visualization**: Live tracking of file processing through perception system stages

## Safety and Control Systems
- **Budget Management**: Token and resource usage tracking with configurable limits
- **Component Status Monitoring**: Real-time health checks for all system components
- **Proactivity Control**: Harmonic score-based messaging with redundancy prevention
